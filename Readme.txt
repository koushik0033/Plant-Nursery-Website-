How to run the Go Green(Plant Nursery management system) using PHP and MySQL

1. Download the project zip file

2. Extract the file and copy GG folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/Html)

4.Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with the name  php_user

6. Import php_user.sql file(given inside the zip package in SQL file folder)

7. Run the script http://localhost/GG/service/customer/project/welcome.php (For Customer Side)

8. And for Admin Side run the script http://localhost/GG/service/admin/admin_login.php

Note: IF ANY WARNING ISSUE BEEN SEEN AFTER LOGGIN IN OR WHILE VIEWING USER PROFILE JUST REFRESH THE PAGE.
